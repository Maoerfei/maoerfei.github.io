    • 文学理论--绪论

    一. 何谓理论

    理论≈哲学
    即：对现象的抽象分析和概括

    二. 理论需要的能力

    1. 深入思考的能力（反思 批判）

    2. 分析现象背后本质（解读 阐释）

    3. 乔纳森 卡勒

    1. 理论是分析的话语
    2. 理论是对常识的批评
    3. 理论具有反射性（反思性）是关于思维的思维
    4. 理论是跨学科的

    三. 何谓文学理论

    1. 文学的“哲学”

    1. 作者之意与读者之意
    2. 你是否看到一只“鸡”？
    3. 随意解读（基于文本）与科学解读
    4. 阅读为何：A接受;B反思

    2. 批判作品背后价值系统

    3. 提供解读作品角度、方法

    4. 跨学科性：

    1. 哲学
    2. 心理学
    3. 语言学
    4. 社会学
    5. ......

    四. 文学理论的研究对象

    1. 文学四要素《镜与灯》

    2. 文学理论研究的五个方面

    1. 文学本质论
    2. 文学活动论
    3. 文学创作论
    4. 文学文本论
    5. 文学接受论

    五. 反思、常识、分析阐释、实践

    1. 反思是理论的一种品格

    反思即事后思维⇒批判

    2. 理论与常识的关系

    共同点：背后都是一套观念体系
    常识：源于经验，而非证明
    理论：揭示常识背后隐含的道理
    区别：理论是经过过渡的观念体系

    3. 分析阐释是当代理论品格

    传统理论：发现本质、揭示规律⇒人文特点、人（bz人非理性动物）
    现代理论：分析阐释

    4. 理论与实践的关系

    1. 理论源于实践⇒多元、丰富

    2. 实践：物质与精神
